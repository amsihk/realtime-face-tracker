# Simple Face Tracker

A Python program that tracks faces in real-time using a webcam and the MediaPipe Tasks API. 
The program automatically downloads the required AI face detection model on its first run.

## Requirements
Install the necessary libraries via your terminal:
`pip install opencv-python mediapipe`

## How to Run
Navigate to the `realtime-face-tracker` folder and run the following command in your terminal:
`python main.py`

(Press the 'q' key to close the camera window).
